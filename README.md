## How to run the program

Make sure to have C++ compiler installed inside the local computer and compile the cpp file
```bash
# Method 1
g++ main.cpp

# Method 2
gcc main.cpp
```

Execute the .exe file on Windows OR .out on MacOS
```bash
# Windows
./a.exe

# Macos
./a.out
```

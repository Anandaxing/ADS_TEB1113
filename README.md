# Algorithm & Data Structure TEB1113

This github repository is created with the purpose of storing ADS Lab assignments and tasks.

## How to run the program

Make sure to have C++ compiler installed inside the local computer and compile the cpp file
```bash
g++ main.cpp
// or
gcc main.cpp
```

Execute the .exe file on Windows and .out on MacOS
```bash
./a.exe
// or
./a.out
```
